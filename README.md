# Music Library API

This project is a RESTful API built using Spring Boot that allows users to manage songs in a music library.

## Features
- Get all songs
- Get song by ID
- Add a new song
- Update song details
- Delete a song

## Technologies Used
- Java
- Spring Boot
- Maven
- REST API

## Endpoints
GET /songs  
GET /songs/{songId}  
POST /songs  
PUT /songs/{songId}  
DELETE /songs/{songId}
